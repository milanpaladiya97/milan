== Description ==
Preloader Animation plugin is very helpful for you if you find an animation add before loading your website. Here you can add animation easily before loading page. You can customize the animation and background color.

== Installation ==
1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. After installing you can see "Dashboard" a new menu "Page Loading Animation".
